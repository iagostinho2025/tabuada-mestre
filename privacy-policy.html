Politica de Privacidade

Aplicativo: Tabuada Mestre

Data de vigencia: 31/03/2026

1. Coleta de dados

O Tabuada Mestre nao coleta dados pessoais identificaveis (nome, email, telefone, documento ou localizacao precisa).

2. Dados armazenados no dispositivo

O aplicativo salva apenas progresso do jogo localmente no proprio dispositivo (pontuacoes, historico e configuracoes).

Se o aplicativo for desinstalado ou se os dados forem limpos, esse conteudo pode ser perdido.

3. Anuncios e rastreamento

Esta versao do aplicativo nao utiliza SDK de anuncios de terceiros e nao faz rastreamento comportamental.

4. Compartilhamento de dados

Nao ha compartilhamento de dados pessoais com terceiros.

5. Publico infantil

O aplicativo possui conteudo educacional. Nao ha compras in-app nem coleta de dados pessoais para fins comerciais.

6. Contato

Para duvidas sobre privacidade, disponibilize um canal oficial (email ou formulario) na pagina publica desta politica.

Link publico desta politica: https://tabuadamestre.mibdigital.com.br/privacy-policy.html
